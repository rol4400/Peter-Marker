Peter Marker Extension Test

Testing Instructions

Color Picker Touch Test:

  1. Click the pen icon to activate drawing mode
  2. Click/tap the color button (palette icon) to open color picker
  3. Try clicking/tapping outside the color picker - it should close
  4. Test both with mouse clicks and touch events

Eraser Visual Feedback Test:

  1. Activate drawing mode and draw some lines
  2. Click the eraser button to enable eraser mode
  3. Move cursor/finger over canvas - you should see a glassy circle
  4. Start erasing - the circle should follow your movement
